## Overview
Exploratory Data Analysis (EDA) on Netflix movies and TV shows dataset to
identify trends in content type, release year, country, genre, and ratings.

## Tools & Technologies
- Python
- Pandas, NumPy
- Matplotlib, Seaborn
- Jupyter Notebook

## Key Insights
- Movies dominate Netflix content compared to TV shows
- Significant growth in content after 2015
- United States is the largest content producer
- Drama and International Movies are most common genres
